❓ Guía rápida

        

          

            Agregar un libro nuevo

            Pestaña "Agregar libro" → llena los datos → clic "Generar código" → copia el código → pégalo en js/libros.js en GitHub antes del comentario final → Commit changes.
          

          

            Cambiar el estado de un libro

            GitHub → js/libros.js → lápiz ✏️ → busca el código del libro (ej: JUV-03) → cambia estado: "D" por el nuevo estado → Commit changes.
          

          

            Marcar como no disponible

            En js/libros.js, busca el libro → cambia disponible: true a disponible: false.
          

          

            Estados: A = Excelente · B = Bueno · D = Deteriorado · C = Crítico
          

          

            Categorías: AUT · CIE · CLA · JUV · MIS · NAR · VAR
